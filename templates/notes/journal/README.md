# Journal

Session journal entries capture what changed in one working session.

Create one entry per session, named with local time:

```text
YYYY-MM-DD-HHMM.md
```

Use entries for:

- Context that shaped the session
- Decisions made or discoveries found
- Open questions to revisit later
- Mood or working state, when it helps future recall

Don't use entries for:

- Secrets, tokens, credentials, or private keys
- Raw transcripts or pasted chat logs
- Personal data that doesn't belong in agent memory
- Generated filler notes with no real session signal
